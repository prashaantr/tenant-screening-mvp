# TenantScreen AI - MVP Landing Page

Landing page for tenant screening SaaS validation (Linear: PRA-93).

## Live Site

Deploy to Vercel and the site will be available at your Vercel URL.

## Features

- 🏠 Professional landing page targeting property managers
- 📝 Waitlist form capturing company info and pain points
- 📱 Fully responsive design
- ⚡ Static site - no build required

## Deploy to Vercel

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import this repository
3. Deploy (no configuration needed)

## Form Setup

The waitlist form uses Formspree. To activate:

1. Create account at [formspree.io](https://formspree.io)
2. Create new form
3. Replace `action="https://formspree.io/f/placeholder"` in `index.html` with your form URL

## Target Metrics

- 50 waitlist signups in 2 weeks
- 20%+ conversion → proceed with full MVP
